# Placement and Training Cell Website

## Overview

The Placement and Training Cell (PTC) website is a digital platform designed to streamline the placement process at IIIT Vadodara. It offers a centralized location for students to register, apply for jobs, and receive placement-related notifications. Placement officers can manage job postings, track student applications, and perform various administrative tasks. The system ensures data security, scalability, and accessibility.

## Features

- **User Registration and Authentication:** Students and placement officers can securely register and log in using their institutional email IDs.
- **Role-Based Access Control (RBAC):** Three user roles with different levels of access: Admin, Placement Officer, and Student.
- **Job Posting Management:** Placement officers can post job openings with detailed descriptions and eligibility criteria.
- **Student Application Tracking:** Students can apply for jobs and track their application status.
- **Gallery Management:** Admins can upload and manage placement event images and success stories.
- **Automated Notifications:** Students receive timely notifications about job openings, deadlines, and updates.
- **Administrative Dashboard:** Admins get a comprehensive view of the system’s usage, including placement statistics and user engagement.
- **Search and Filter:** Advanced job search and filter options based on criteria such as location, salary, and job type.
